# Sales Analysis (SQL + Python)

## Objective
Analyze an e-commerce sales dataset to answer business questions about revenue, products, customers, and trends.

## Tech Stack
- SQL (SQLite)
- Python (pandas, matplotlib)
- Jupyter Notebook

## Repository Structure
- `data/`: raw and processed data (not included if too large)
- `sql/`: SQL queries
- `notebooks/`: analysis notebooks
- `images/`: figures for the README
- `src/`: helper scripts (ETL, utilities)
